# GrandVista - Luxury Hospitality & Hotel Booking Frontend

GrandVista is an elite, multi-page frontend interface designed specifically for premium hotels, luxury resorts, and boutique hospitality services. This responsive web template leverages micro-frameworks alongside structural UI components to deliver an immersive end-user experience, optimized for conversion and booking workflows.

---

## Architecture & System Structure

The workspace contains specialized layouts tailored to modern booking management ecosystems:
* **Landing Hub (`index.html` / `index-two.html`):** Offers high-impact homepage variants utilizing advanced layout animations and video players.
* **Reservation Blueprint (`booking.html`):** Provides a robust frontend interface wireframe optimized for scheduling and check-in parameter fields.
* **Property Showcases (`accomodation.html` / `room-details.html` / `gallery.html`):** Responsive grid systems designed to render high-resolution assets, room features, and rate configurations.
* **Operational Overviews (`about-us.html` / `staff.html` / `contact-us.html`):** Context-ready presentation frameworks for executive teams, support channels, and interactive geographical forms.
* **Marketing Panels (`blog.html` / `single-blog.html`):** Content layouts ready to handle search engine indexing and brand storytelling.

---

## Styling & Technical Composition

* **Core Structural Frameworks:** Bootstrap v3/v4 Ecosystem (`bootstrap.min.css`) combined with UIkit Core (`uikit.min.css`).
* **Interactive Animation Systems:** Animate.css for entrance animations and Revolution Slider configurations (`rev-slider/settings.css`).
* **Visual Components:** Owl Carousel 2 (`owl.carousel.css`, `owl.video.play.html`) alongside LightSlider implementations for responsive media rendering.
* **Data Selection Frameworks:** Embedded localized datepicker UI models (`datepicker.css`).
* **Typography & Icons:** Multi-family Google Font integrations (Playfair Display, Karla, Lora) coupled with FontAwesome v4 asset libraries.
